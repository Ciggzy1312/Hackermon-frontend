A bit about me

Hi, I am Hackermon a Discord Bot for every hackathon servers out there. I help out students by letting them know about many upcoming hackathons and encouraging them to take part in order for them to explore and grow.
Want to talk more with me?
Then check out my commands and invite me to your server

Commands List :

!help

Help command will give you all the information about Hackermon which includes all the commands and what they do and the prefix needed to talk to Hackermon

!link <@URL>

Will let the user post the mentioned URL but only if he has the 'Moderator' role in the server to avoid spamming

!kick <@user>

Will kick the mentioned user out of the server

!ban <@user

Will ban the mentioned user from the server

!mlh

Will show you all the upcoming hackathons organized by MLH

!hackclub

Will show you all the upcoming hackathons organized by HackClub

!devfolio

Will show you all the upcoming hackathons organized by Devfolio
